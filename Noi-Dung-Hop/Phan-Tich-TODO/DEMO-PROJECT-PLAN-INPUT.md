# Demo Project Plan Input

## Dự án
PM Agent

## Mục tiêu
Xây dựng một PM Agent có thể hỗ trợ tạo tài liệu PM, làm rõ yêu cầu, tạo biên bản họp, bóc action items và xuất PDF qua Telegram.

## Phạm vi bước đầu
- Requirement clarification
- Meeting minutes
- Action items
- Status report
- PDF export

## Ràng buộc
- Output cuối mặc định là PDF
- Phải hỗ trợ tiếng Việt
- Mỗi skill phải có demo review được
