# Demo Action Items Input

## Chủ đề
Họp triển khai PM Agent giai đoạn đầu

## Nội dung
- Cần hoàn tất skill `markdown-to-pdf` trước.
- Sau đó phải build `requirement-clarifier` và `meeting-minutes-generator`.
- Mỗi skill phải có file demo và file PDF để review.
- Cần tiếp tục tạo các skill còn lại trong bộ tối thiểu.
- Chưa chốt thứ tự đầy đủ cho toàn bộ các skill còn lại.
- Về lâu dài phải có tài liệu gửi khách hàng và cấp trên.
