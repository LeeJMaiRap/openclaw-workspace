# Project Plan - PM Agent

## 1. Mục tiêu dự án
- Xây dựng một PM Agent có thể hỗ trợ các tác vụ PM cốt lõi và xuất tài liệu PDF để review qua Telegram.

## 2. Phạm vi
### Trong phạm vi
- Requirement clarification
- Meeting minutes
- Action items
- Status report
- PDF export

### Ngoài phạm vi
- Voice workflow đầy đủ
- Hợp đồng pháp lý cuối cùng
- Dashboard phức tạp

## 3. Giai đoạn triển khai
- Giai đoạn 1: Xây skill nền export PDF
- Giai đoạn 2: Xây skill PM cốt lõi
- Giai đoạn 3: Chuẩn hóa workflow và review tài liệu

## 4. Milestones chính
- Hoàn tất `markdown-to-pdf`
- Hoàn tất 4 skill PM lõi đầu tiên
- Hoàn tất bộ 9 skill tối thiểu

## 5. Phụ thuộc và rủi ro chính
- Phụ thuộc vào renderer PDF ổn định và font Unicode tiếng Việt
- Rủi ro tạo quá nhiều skill trước khi chốt đúng artefact thực tế

## 6. Bước tiếp theo
- Hoàn tất demo cho toàn bộ bộ skill tối thiểu
- Review output thực tế theo từng loại tài liệu PM
