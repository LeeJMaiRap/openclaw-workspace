# Demo Requirement Input

Cần xây dựng một PM Agent dùng qua Telegram.
Agent này nên hỗ trợ công việc PM thực tế như ghi chú cuộc họp, tạo tài liệu PM, quản lý tri thức dự án và follow-up công việc.
Về lâu dài có thể hỗ trợ voice.
Chưa rõ nên ưu tiên tính năng nào trước, và chưa rõ output tài liệu chính cần là gì ngoài việc muốn có file PDF để dùng thực tế.
