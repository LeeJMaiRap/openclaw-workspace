# Demo Markdown to PDF

## Mục tiêu
Tài liệu này dùng để test skill `markdown-to-pdf` theo final spec mới.

## Nội dung kiểm tra
- Xuất được file PDF thật
- Hiển thị tiếng Việt đúng dấu
- Có hỗ trợ style xuất bản
- Dùng được cho tài liệu PM thực tế

## Ví dụ nội dung PM
**Dự án:** PM Agent qua Telegram  
**Trạng thái:** Đang phân tích lại từ đầu  
**Output cuối mặc định:** PDF

## Danh sách kiểm tra
- [x] Có file nguồn `.md`
- [x] Có file output `.pdf`
- [x] Có font Unicode tiếng Việt
- [x] Có renderer baseline và styled

## Ghi chú
Nếu anh đọc được tiếng Việt bình thường trong file PDF này thì skill đã đạt bước đầu.
