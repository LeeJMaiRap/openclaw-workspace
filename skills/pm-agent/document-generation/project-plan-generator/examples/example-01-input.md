# Ví dụ 1: Input cho project plan hệ thống quản lý task

## Nguồn: Charter + Requirements + Spec

### Từ Charter
- Dự án: Hệ thống quản lý task nội bộ
- Timeline: khoảng 2 tháng
- Team nội bộ 5 người
- Phạm vi MVP tập trung vào quản lý task cốt lõi

### Từ Requirements
- Tạo task
- Phân công người thực hiện
- Theo dõi trạng thái task
- Comment trong task
- Đính kèm file
- Phân quyền admin/member
- Responsive UI

### Từ Spec
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT
- Không có tích hợp bên ngoài trong MVP

### Ràng buộc
- Chỉ dùng nội bộ
- Không cần import từ Trello
- Cần hoàn thành MVP trước khi nghĩ đến dashboard nâng cao
