# Ví dụ 1: Ghi chú họp kickoff dự án web bán hàng

Họp kickoff dự án web bán hàng
- Khách hàng muốn có giỏ hàng
- Thanh toán online
- Quản lý sản phẩm
- Báo cáo doanh thu
- Anh Minh hỏi về tích hợp kho
- Chưa rõ API kho
- Timeline: 3 tháng
- Budget: thương lượng sau
- Họp lại tuần sau
- Cần làm rõ yêu cầu chi tiết
- Phân công: Lan làm frontend, Minh làm backend
- Dùng React + Node.js
