# tmp

Scratch tạm cấp workspace.

## Quy tắc

- Có thể xoá định kỳ.
- Không lưu source framework, project artifact dài hạn, hoặc output cần giữ ở đây.
- Nếu là export/runtime output, chuyển sang `ops/exports/`.
- Nếu là script/tooling, chuyển sang `ops/scripts/`.
- Nếu là dữ liệu project, chuyển sang `projects/` theo lifecycle.

Hiện chỉ giữ skeleton và `.gitkeep` cần thiết.
