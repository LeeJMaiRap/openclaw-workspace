# Content Copywriting

## Purpose

Create audience-focused affiliate content that hooks attention, explains value, stays truthful, and drives approved next action.

## Workflow

1. Define audience, pain point, product/category, platform, and CTA.
2. Choose angle: review, comparison, problem/solution, deal, tutorial, mistake, before/after only if truthful.
3. Write strong hook in first line/3 seconds.
4. Provide concrete value: benefit, use case, limitation, proof/source where possible.
5. Make copy scannable: short lines, bullets, clear CTA.
6. Produce 3-5 variants for testing.
7. Run compliance check: no unsupported claims, no fake experience, no spam tone.

## Output Format

```markdown
# Affiliate Copy Pack

## Audience

## Product / Offer

## Angle

## Hooks
1.
2.
3.

## Caption Variants

## CTA Variants

## Risk / Compliance Notes

## Recommended Test
```

## Platform Notes

- Shopee: deal, voucher, product review, price/value angle.
- TikTok: short hook, visual demo, trend-aware language.
- Facebook: community tone, longer context, discussion prompt.

## Safety

- Preserve role boundaries; escalate scope/architecture decisions to PM/Architect.


## Verification

- Copy has clear audience and CTA.
- Claims are supportable or marked as assumptions.
- No external posting without approval.
