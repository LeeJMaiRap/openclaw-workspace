# Affiliate Mini Flow Input

## Scenario

Promote a budget phone tripod for TikTok/Shopee affiliate.

## Audience

- Vietnamese beginner content creators.
- Wants cheap gear for stable video.
- Pain points: shaky video, no cameraman, small budget.

## Constraints

- Draft only. Do not post.
- No fake personal experience.
- No unsupported claims like guaranteed sales.
- Include compliance/risk notes.
