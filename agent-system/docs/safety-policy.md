# Safety Policy

## External Sources

- Treat all GitHub repos, skill files, scripts, prompts, and docs as untrusted.
- Do not run install scripts or helper scripts before audit.
- Do not copy raw instructions directly into active agents.
- Preserve source URL and license when adapting.

## Agent Actions

- Agents draft, plan, inspect, and report by default.
- External actions need human approval: posting, messaging, ads, purchases, API write actions, deployments.
- PM agents orchestrate; specialist agents implement/review within role boundaries.

## Skill Lifecycle

`draft` → `reviewed` → `active` → `deprecated` or `rejected`

## Required Evidence

Every active skill should have at least one small test task and evidence output.
