# Voice Script Guide

Hướng dẫn biến text thành script dễ nghe.

## Khi nào nên dùng voice

Phù hợp:
- tóm tắt ngắn
- nhắc việc
- cập nhật trạng thái nhanh
- giải thích ngắn gọn

Không phù hợp:
- bảng biểu dài
- nhiều bullet phức tạp
- API spec / code / config
- nội dung cần copy/paste chính xác

## Quy tắc viết script dễ nghe

### 1. Viết như đang nói với người thật
- câu ngắn hơn
- ít cấu trúc cứng
- tránh nhồi quá nhiều ý trong một câu

### 2. Ưu tiên takeaway trước
Nói điều quan trọng nhất trước, rồi mới đến chi tiết hỗ trợ.

### 3. Đọc deadline và action item rõ ràng
Nếu có:
- deadline
- việc cần làm
- cảnh báo

thì phải giữ nguyên và nói thật rõ.

### 4. Nếu text quá dài, rút gọn
Không cần đọc nguyên văn nếu điều đó làm audio khó nghe.

## Ví dụ

### Text gốc
"Có 3 việc chính: chốt API contract trước 17/05, review dashboard scope, và chuẩn bị checklist validation cho task form."

### Voice script tốt
"Anh ơi, có ba việc chính cần nhớ. Một là chốt API contract trước ngày mười bảy tháng năm. Hai là review lại scope của dashboard để tránh nở phạm vi. Ba là chuẩn bị checklist validation cho task form."

### Voice script không tốt
"Có 3 việc chính: chốt API contract trước 17 slash 05, review dashboard scope, và chuẩn bị checklist validation cho task form."

Vấn đề:
- quá sát text viết
- nghe cứng
- đọc ký hiệu ngày không tự nhiên

## Khi nào nên chia nhiều audio
- quá 1-2 phút
- có nhiều phần khác nhau
- người nghe cần tiêu hóa từng phần riêng

## Kết luận
Voice tốt không phải là đọc nguyên văn text. Voice tốt là giúp người nghe hiểu nhanh hơn, nhớ tốt hơn và đỡ mệt hơn khi tiếp nhận nội dung.
