# Ví dụ 1: Input cho requirements document hệ thống quản lý task

## Nguồn: Requirement Clarification + Charter

### Từ Charter
- **Mục tiêu:** Xây dựng hệ thống web nội bộ giúp team quản lý task, phân công người thực hiện, theo dõi tiến độ
- **Phạm vi in-scope:**
  - Tạo task
  - Phân công người thực hiện
  - Theo dõi trạng thái/tiến độ task
  - Comment trong task
  - Đính kèm file
  - Đặt deadline
  - Phân quyền admin / member

### Từ Requirement Clarification
- Team 5 người
- Cần thay thế Trello
- Chỉ dùng nội bộ
- Không cần import dữ liệu từ Trello
- Cần comment, file đính kèm, deadline
- Cần phân quyền admin / member

### Yêu cầu phi chức năng đã xác nhận
- Responsive (mobile + desktop)
- Bảo mật thông tin người dùng
- Giao diện đơn giản, dễ dùng
