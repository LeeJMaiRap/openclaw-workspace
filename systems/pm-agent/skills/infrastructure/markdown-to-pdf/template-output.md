# Markdown to PDF Output

## Input
- **Source markdown:** [path/to/file.md]
- **Output pdf:** [path/to/file.pdf]
- **Paper size:** [A4/Letter]
- **Notes:** [Optional notes]

## Expected Result
- File PDF được tạo thành công
- Nội dung đọc được, không lỗi encoding cơ bản
- Heading, bullet list, đoạn văn có bố cục dễ đọc
- Có thể dùng để chia sẻ / trình bày / lưu trữ

## Verification Checklist
- [ ] File PDF đã được tạo
- [ ] Mở được file PDF
- [ ] Nội dung không bị rỗng
- [ ] Tiêu đề và heading hiển thị ổn
- [ ] Danh sách / bảng quan trọng không bị vỡ nghiêm trọng
- [ ] Tên file đầu ra đúng như mong muốn
