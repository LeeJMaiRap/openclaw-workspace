# Ví dụ 1: Input cho lessons learned hệ thống quản lý task

## Nguồn: Issue log + status dashboard + ghi chú thực thi

### What went well (quan sát được)
- WBS rõ nên chuyển sang task board nhanh
- Task board giúp nhìn blocker rõ
- PM Agent chuẩn hóa tài liệu nhanh và đều

### What could be improved
- Thiếu design direction làm chặn frontend
- Validation deadline bị bỏ sót đến lúc test
- Một số ước lượng effort còn hơi lạc quan

### Issues đã gặp
- Thiếu design direction cho layout
- Form create task thiếu validation deadline
- Lỗi version dependency đã từng chặn install

### Dashboard observation
- Project ở trạng thái At Risk do variance +2 days
- Scope completion 37.5%
- Forecast effort tăng nhẹ

### Recommendation sơ bộ
- Chốt design direction sớm hơn
- Test validation sớm hơn
- Review effort estimation kỹ hơn trước execution
