# Reference: Anthropic Frontend Design Skill

Source: https://www.skills.sh/anthropics/skills/frontend-design

## Summary

This skill guides creation of distinctive, production-grade frontend interfaces and avoids generic AI-looking UI.

Useful ideas:

- understand purpose and audience before coding
- choose a clear aesthetic direction
- make UI memorable and intentional
- implement real working code, not static mockups
- maintain production-grade quality
- refine details: layout, spacing, typography, states, interaction

## Best Fit In Agent-Teams

### UX/Design Agent

Use for:

- design direction
- tone/aesthetic selection
- user flow and screen intent
- memorable visual concept

### Frontend Agent

Use for:

- production-grade UI implementation
- cohesive visual execution
- detail refinement
- avoid generic components unless project requires plain style

### Code Review Agent

Use for:

- UI quality review
- detecting generic/low-effort frontend output

## What Not To Copy Blindly

- extreme aesthetics when product needs plain utility
- visual ambition that hurts usability/accessibility
- design changes outside approved product scope

## OpenClaw Rule

Frontend design direction must be chosen intentionally and recorded before large UI implementation.
