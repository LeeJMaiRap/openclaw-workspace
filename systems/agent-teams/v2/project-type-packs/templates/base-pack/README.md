# Project Type Pack Template

## Purpose

Use this template to create an Agent-Teams v2 project type pack.

A pack is an add-on to the v1 starter and v2 governance. It does not replace PM orchestration, dependency governance, browser evidence rules, security/performance gates, or deploy gates.
