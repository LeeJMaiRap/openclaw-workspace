# Phase 31 — Clean-Slate Mini Local Project Scope

## Purpose

Select and define the first Agent-Teams v1 real-project pilot scope using a clean-slate mini local project.

## Decision

Use a small local-only project called `mini-issue-tracker-v1`.

Clean slate avoids legacy project risk and makes Agent-Teams orchestration quality easier to measure.

## Pilot Type
