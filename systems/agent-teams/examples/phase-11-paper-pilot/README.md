# Phase 11 Paper Pilot — Issue Tracker MVP

## Status

Paper pilot only. No application code built.

## Goal

Simulate PM Agent using Agent-Teams on a small issue tracker MVP through task packets, paper reports, gates, and final handoff.

## Pilot Scope

MVP:

```text
- landing page
- issue list
- issue detail
- create issue
- update issue status
- simple owner/admin view
- local/mock auth decision only
```

Out of scope:

```text
- production deploy
- real users/secrets
- payment
- notifications
- team collaboration
- file attachments
```
